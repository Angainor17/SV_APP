# Исходное ТЗ: Новые экраны Wiki-модуля

## Источник

Архитектура нового экрана должна быть аналогичной с использованием ViewModel, UseCase, Hilt, Compose.

---

## 1. Иконка избранного на RootWiki

На экране RootWiki справа от поля поиска необходимо показать иконку "Избранное" (сердечко).

**Требования:**
- Иконка отображается только в том случае, если в локальном хранилище есть данные в избранном
- При нажатии открывается экран "Избранное"
- Иконка плавно исчезает с анимацией при очистке избранного

---

## 2. Экран "Избранное"

### Компоненты:

1. **Заголовок** "Избранное"
2. **Стрелочка назад** слева от заголовка для возврата на прошлый экран
3. **Иконка очистки** (корзина) в верхнем правом углу
   - При нажатии выводится AlertDialog: "Вы точно хотите очистить избранное?"
   - Кнопки: "Да", "Нет"
   - При "Нет" - диалог закрывается
   - При "Да" - локальное хранилище очищается, экран закрывается, иконка на RootWiki плавно исчезает
4. **Список названий статей** из избранного
   - При нажатии на пункт открывается экран "Статья"

---

## 3. Экран "Статья"

### Компоненты:

1. **Заголовок** - название статьи
2. **Стрелочка назад** слева от заголовка для возврата
3. **Текст статьи** с вертикальным скроллом и возможностью выделения текста, а также переходами по ссылкам
4. **Кнопки действия** в правом верхнем углу:
   - **Избранное** (сердечко):
     - При нажатии статья удаляется из избранного
     - Экран не закрывается
     - При возврате на "Избранное" пункт удаляется с плавной анимацией
     - Можно нажать повторно, чтобы вернуть в избранное
   - **Внешняя ссылка** (новая иконка):
     - При нажатии открывается Intent с URL статьи
     - Пример: для "Марксизм" → "https://svremya.su/Марксизм"
     - URL должен сохраняться в БД при первом поиске
